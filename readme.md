# D&D Friendly

App web pour aider les joueurs et le MJ à gérer une campagne
de Donjons & Dragons, en présentiel ou à distance.

## Fonctionnalités
- Gestion de campagnes.
- Profils de personnages.
- Jets de dés avec historique.
- Mode campagne physique / virtuelle.

## Stack
- Node.js / Express
- EJS (server-side rendering)
- PostgreSQL (à venir)

## Installation
npm install
