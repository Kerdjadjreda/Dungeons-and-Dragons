## Roadmap

### Phase 1 – MVP
- Mise en place du serveur (Node.js / Express)
- Création des routes CRUD
- Rendu côté serveur avec EJS
- Gestion des données en mémoire / JSON

### Phase 2 – Persistance
- Intégration d’une base de données
- Remplacement des données temporaires par une persistance réelle

### Phase 3 – Front dynamique
- Frontend plus interactif (React ou équivalent)
